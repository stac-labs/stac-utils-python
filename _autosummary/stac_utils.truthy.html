stac_utils.truthy

stac_utils.truthy(thing: str, true_values: Optional[list] = None) → bool

Evaluates whether a string is True. All of the following will evaluate as True: 1, Yes, Y, Yeah, T, True, true. Most other things will evaluate as False, including None and undefined.

Parameters:

  • thing – Item to evaluate

  • true_values – Optional list of alternate True values

Returns:

True if a truthy value, False if anything else
