stac_utils.listify

stac_utils.listify(string: str, type_: type = <class 'str'>, sep: str = ', ', ignore_errors: bool = False, ignore_empty: bool = True) → list

Split a string into a list, converting to a type as necessary. For example, “foo, bar, spam” will become [“foo”,”bar”,”spam”].

Parameters:

  • string – String to split into list

  • type – Type to convert, if desired, str by default

  • sep – Separator for list

  • ignore_errors – False by default

  • ignore_empty – True by default

Returns:

String converted to list
